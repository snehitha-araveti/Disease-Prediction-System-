# Disease Prediction System (Java OOP Case Study)

A comprehensive **Java-based Disease Prediction System** developed using **Object-Oriented Programming (OOP)** concepts.  
This project analyzes symptoms entered by the user, predicts possible diseases, provides treatment recommendations, stores diagnosis history, and generates medical reports.

---

# Project Overview

The Disease Prediction System is a console-based healthcare application designed to simulate a real-world medical diagnosis workflow.

The system allows users to:

- Predict diseases using symptoms
- Browse disease information
- Save diagnosis history
- View previous records
- Receive treatment suggestions
- Generate medical reports
- Check emergency symptoms
- Provide feedback for previous diagnosis

This project demonstrates strong implementation of **core OOP principles**, modular architecture, and file-based data persistence.

# Features
